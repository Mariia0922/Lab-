## Хід виконання роботи

Я створила програму мовою Python відповідно до свого варіанта (номер списку: 14).

С11: 3.

За отриманим значенням я визначила клас морський човен та для цього створила клас Boat, який містить п'ять полів: назву, довжину, швидкість, кількість членів екіпажу та рік випуску.
Після цього я створила список об'єктів класу Boat та заповнила його даними. 
Далі за допомогою стандартного методу sort() я відсортувала список за довжиною човна за зростанням, а при однаковій довжині, то за швидкістю за спаданням.
Потім я реалізувала пошук об'єкта за назвою. Для цього за допомогою циклу `for` перебрала всі елементи списку та порівняла їх із заданою назвою. 
Якщо потрібний об'єкт було знайдено, програма виводить його характеристики на екран.
Також я використала конструкцію `try-except` для обробки можливих помилок. 
У результаті програма виводить відсортований список човнів та інформацію про знайдений об'єкт.
