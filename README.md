## Хід виконання роботи

Я створила програму мовою Python для обчислення подвійної суми за формулою мого варіанта (номер списку: 14).
С2: +,
С3: 2,
С5: -,
С7: byte.
Спочатку я реалізувала введення значень a, b, n та m з клавіатури, щоб користувач міг самостійно 
задавати межі сумування. Для обчислення результату я використала два вкладені цикли for, які перебирають усі можливі значення індексів i та j у заданих межах. 
На кожній ітерації я обчислювала значення виразу (i−j)/(i+2) та додавала його до загальної суми.

Також я реалізувала обробку можливих помилок за допомогою конструкції try-except. 
Зокрема, я передбачила перевірку коректності введених даних і випадок ділення на нуль, 
який може виникнути при значенні i = -2. Після завершення всіх обчислень програма виводить 
отриманий результат на екран. Завдяки цьому програма коректно виконує обчислення та обробляє 
можливі помилки під час роботи.
Для варіанта 14 використовувалась формула:

`S = ΣΣ ((i - j) / (i + 2))`
