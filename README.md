## Хід виконання роботи

Я створила програму мовою Python для обробки тексту відповідно до мого варіанта (номер списку: 14).

С3: 2 (StringBuffer),

С17: 14 (видалити всі наступні входження першої літери кожного слова).

Спочатку я реалізувала введення тексту з клавіатури, щоб користувач міг самостійно задавати рядок для обробки. 
Також я додала перевірку на порожній рядок, щоб уникнути некоректної роботи програми.

Для обробки тексту я використала метод split(), який розбиває введений рядок на окремі слова. 
Далі за допомогою циклу for я послідовно опрацьовувала кожне слово. У кожному слові я залишала першу літеру без змін, а всі наступні входження цієї літери видаляла. 
Після цього оброблені слова додавалися до нового тексту.

Також я реалізувала обробку можливих помилок за допомогою конструкції try-except. Після завершення обробки програма виводить отриманий результат на екран. 
Завдяки цьому програма коректно виконує поставлене завдання та обробляє можливі помилки під час роботи.

## Висновки
Під час виконання лабораторної роботи я ознайомилася з основними способами роботи з рядками в мові програмування Python. 
Я навчилася розбивати текст на окремі слова, опрацьовувати їх за допомогою циклів та формувати новий рядок відповідно до поставленого завдання.
Найбільші труднощі виникли під час реалізації алгоритму видалення повторних входжень першої літери в кожному слові та обробки можливих помилок. 
У результаті виконання роботи я закріпила навички роботи з рядками, циклами та конструкцією try-except.
