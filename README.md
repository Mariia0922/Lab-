## Хід виконання роботи

Для виконання лабораторної роботи я використала варіант 14.

Спочатку я визначила значення залишків від ділення номера варіанта:

C5 = 4;
C7 = 0;
C11 = 3.

Відповідно до отриманих значень було визначено, що необхідно виконати множення матриць A × B, 
використати тип елементів double та обчислити суму найбільших елементів кожного рядка результуючої матриці.
У програмі я створила дві матриці A та B, після чого реалізувала функцію множення матриць. 
Для обчислення елементів результуючої матриці C я використала вкладені цикли for. 
Після отримання матриці C я знайшла найбільший елемент у кожному її рядку та обчислила їх суму.

Також у програмі реалізовано перевірку можливості множення матриць та обробку помилок за допомогою конструкції try-except. 
Результати роботи програми виводяться на екран у зручному вигляді.
