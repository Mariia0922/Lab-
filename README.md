## Хід виконання роботи

Я створила програму мовою Python відповідно до свого варіанта (номер списку: 14).
С2: 0 (List),
С3: 2 (двозв'язний список).
Для виконання роботи я використала класи із попередньої лабораторної роботи. 
Після цього я створила клас Node, який використовується для зберігання елементів двозв'язного списку.
Також я створила клас SweetCollection, у якому реалізувала додавання, видалення та виведення елементів колекції. 
Далі я створила кілька об'єктів солодощів і додала їх до списку.
Після додавання елементів програма виводить вміст колекції на екран. 
Потім один із елементів видаляється, і колекція виводиться повторно для перевірки результату.
Для обробки можливих помилок я використала конструкцію try-except. 
Завдяки цьому програма коректно виконує всі необхідні операції з колекцією.

Найскладніше було зрозуміти, як працюють посилання на попередній і наступний елементи списку та як правильно реалізувати видалення елементів.
